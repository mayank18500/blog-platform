# 📝 Blog App

A simple blog application built using **Node.js**, **Express.js**, **EJS**, and **CSS**. Users can create, view, edit, and delete blog posts.

## 🚀 Features

- Create new blog posts
- View a list of all blog posts
- Read full content of each post
- Edit existing posts
- Delete posts
- Responsive design with light/dark theme toggle

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js
- **Templating Engine:** EJS
- **Frontend:** HTML, CSS, JavaScript
